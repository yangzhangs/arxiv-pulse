关于 ArxivPulse

ArxivPulse 是一个自动化的学术文章展示平台，专注于追踪和展示 Software Engineering 领域的最新研究成果。

🎯 项目目标

  • 每日自动检索 arXiv 上最新的 Software Engineering 领域论文
  • 通过智能标签系统对论文进行分类和筛选
  • 提供简洁美观的界面，方便快速浏览和搜索
  • 支持飞书消息推送，及时获取最新研究动态

📋 功能特性

  • 自动检索：每周二至周六自动检索最新论文
  • 标签筛选：支持多标签分类和筛选
  • 全文搜索：支持标题、作者、摘要搜索
  • 响应式设计：支持桌面和移动端
  • 消息推送：飞书定时推送整理后的文章
  • 标签扩展：支持动态添加新标签

🔍 检索关键词

Docker CI/CD DevOps 微服务 云原生 Serverless Hugging Face Github Actions Agent skills 弃用包

📬 消息推送

每周二至周六 08:00-09:00（北京时间），系统会自动发送最新论文摘要到飞书。 每篇推送包含：

  • 文章标题和作者
  • PDF 下载链接
  • 50 字以内的简要概述
  • 相关标签
