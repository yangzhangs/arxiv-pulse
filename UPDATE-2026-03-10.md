## ⚠️ 注意事项

### 标签准确性
- 目前采用关键词匹配，可能存在漏标
- 建议定期审查和调整关键词映射表
- 可以在 `src/utils/retag-papers.js` 中添加更多关键词

### 摘要翻译
- 当前为简单关键词替换，非完整翻译
- 如需高质量翻译，需集成翻译 API
- 建议保持英文摘要，前端显示时再做处理

### Arxiv API
- API 可能有速率限制
- 建议在 HEARTBEAT 定时任务中执行检索
- 避免频繁手动调用
