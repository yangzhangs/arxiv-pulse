## 🎯 技术细节

### 摘要解析逻辑

```javascript
formatAbstract(abstract) {
  if (!abstract) return '暂无摘要';
  
  // 双行格式：[EN] xxx\n\n[CN] yyy
  if (abstract.includes('[EN]') && abstract.includes('[CN]')) {
    const parts = abstract.split('\n\n');
    if (parts.length >= 2) {
      const enPart = parts[0].replace('[EN]', '').trim();
      const cnPart = parts[1].replace('[CN]', '').trim();
      return `<div class="text-gray-700 mb-1">${enPart}</div>
              <div class="text-gray-500 text-sm">${cnPart}</div>`;
    }
  }
  
  return abstract;
}
```

### HTML 渲染

```html
<!-- 旧版本 -->
<p x-text="formatAbstract(paper.abstract)"></p>

<!-- 新版本 -->
<div x-html="formatAbstract(paper.abstract)"></div>
```
